# Local PDF Chatbot (RAG)

A local AI-powered PDF chatbot that can read PDFs and answer questions using Retrieval-Augmented Generation (RAG).

## Features

- Reads PDF documents
- Answers questions from PDFs
- Local embeddings
- Vector search
- Offline AI chatbot
- Local RAG pipeline

## Technologies Used

- Python
- Ollama
- LangChain
- ChromaDB
- Phi3 Mini
- Nomic Embed Text
